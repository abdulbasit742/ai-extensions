ARIA Video Capture Assistant
============================

Load this folder with Edge/Chrome -> Extensions -> Developer mode -> Load unpacked.

Use:
1. Open a page containing a video you own or have permission to reuse.
2. Tick "I own/have permission".
3. Click "Detect video" then "Save visible video".
4. If the page uses blob/stream/protected playback, use that site's official download/export first, then use ARIA dashboard "Import latest video".
5. Click "Prepare social video" to create captions and open Facebook, Instagram, LinkedIn, and X draft tabs.

This helper will not bypass DRM, m3u8 streams, watermarks, or site restrictions.
