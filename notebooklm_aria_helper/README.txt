ARIA NotebookLM Video Helper
===========================

Use this extension for NotebookLM video automation.

Install:
1. Open Edge/Chrome extensions page.
2. Turn Developer mode ON.
3. Click Load unpacked.
4. Select the folder that contains this manifest.json directly.

Dashboard:
1. Open ARIA dashboard.
2. Set NotebookLM URL/project keyword and output folder.
3. Click Start NotebookLM video.
4. Solve login/CAPTCHA manually if Google asks.
5. When NotebookLM downloads a video, ARIA moves it to the selected folder.

Safety:
- This extension only runs on https://notebooklm.google.com/*
- It does not touch other tabs.
- If NotebookLM UI changes, ARIA will wait and show the next manual step.
