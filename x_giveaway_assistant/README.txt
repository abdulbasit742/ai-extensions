ARIA X Giveaway Assistant
=========================

What it does:
- Reads the visible X/Twitter giveaway post.
- Extracts handles and likely rules such as follow, repost, like, comment, tag, and notifications.
- Detects exact comment words like BTC, Money, Canva, YES, or Done.
- Detects "must follow me", "follow @name", "tag 3 friends", and "turn on notifications".
- Lets you save an approved friend tag list and randomly chooses from that list when a post says tag friends.
- Highlights Like/Repost/Reply/Follow controls when visible.
- Guide Like and Guide Repost scroll to the exact visible control and highlight it for your confirmation.
- Best Safe Run creates the checklist, copies the comment draft, and guides you to the next required action.
- Next Action moves through Follow, Like, Repost, Comment, and Notifications until all detected requirements are marked done.
- Mark highlighted done records the current manual action in the post checklist.
- Opens required profile links for manual follow.
- Creates a comment draft and can paste it into the reply box without submitting.
- Smart Prepare highlights controls, copies the right comment text, and shows the remaining manual steps.
- Smart Prepare uses only your approved friend list for tag drafts; it never tags random strangers.
- Fill profile post draft opens/fills your X compose box with a profile post draft, but does not submit it.
- Saves a local checklist per post URL.
- The toolbar popup cannot be dragged because the browser controls it. Use "Show bottom floating panel" to reset the dark ARIA panel to the page bottom-right.

Safety:
- This extension does not auto-follow, auto-repost, auto-like, or auto-submit comments.
- It can fill a reply/profile post draft, but public actions must be confirmed by you in the X interface.
- It will not choose random public accounts to tag. Add approved friend handles first.
- Do not store your X password or Groq API key inside a browser extension. If AI analysis is needed, use a local ARIA server proxy so secrets stay on your own PC.
- This protects your account from accidental spam/automation behavior.

Install:
1. Open edge://extensions or chrome://extensions.
2. Turn Developer mode ON.
3. Click Load unpacked.
4. Select this folder:
   x_giveaway_assistant_READY_LOAD_THIS_FOLDER
5. Open an X post and use the ARIA panel or extension popup.

Best use:
1. Open the giveaway post.
2. Click Show bottom floating panel if the dark panel is not visible.
3. Add approved friends, for example: @friend1, @friend2, @friend3.
4. Click Best Safe Run.
5. Click the highlighted public action manually.
6. Click Mark highlighted done.
7. Click Next Action until all detected requirements are complete.
8. Optional: click Fill profile post draft, review, then Post yourself.
