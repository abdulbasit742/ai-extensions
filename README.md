# ARIA AI Extensions

This repository contains the ARIA browser automation extension suite.

## Main extension

- `aria_super_extension/` - ARIA Nexus One Hub, the unified extension with AI chat automation, response capture, Word/Drive bridges, social draft helpers, prompt queues, limit guard, and multi-site controls.

## Helper extensions

- `chatgpt_aria_sender/`
- `chatgpt_image_saver/`
- `chatgpt_word_copier/`
- `clickup_aria_sender/`
- `notebooklm_aria_helper/`
- `social_publisher_assistant/`
- `tiktok_aria_uploader/`
- `universal_aria_sender/`
- `video_capture_assistant/`
- `x_giveaway_assistant/`

## Load in Chrome or Edge

1. Open `chrome://extensions` or `edge://extensions`.
2. Enable Developer mode.
3. Click **Load unpacked**.
4. Select the extension folder you want to run, usually `aria_super_extension`.

Keep API keys and local runtime data outside this repository.
